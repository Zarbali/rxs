# Настройка кнопки «Играть»

При нажатии «Играть» открывается окно с:
- **Скачать пресет** — скачивается файл пресета целиком (Arma 3 Preset RSX_24STS_V_1.html)
- **Запустить Arma 3** — запуск игры через Steam
- **Установить Arma 3** — ссылка в Steam Store

Игрок скачивает пресет, загружает его в Arma 3 Launcher (Моды → Preset → Import). Лаунчер показывает список модов и предлагает подписаться — Steam скачает их автоматически.

---

## Настройка config.js

Файл пресета указывается в `presetUrl`. По умолчанию: `Arma 3 Preset RSX_24STS_V_1.html`

При смене пресета: положите файл в корень проекта и обновите `presetUrl` в config.js.

---

## Ограничения

- Скачивание пресета работает как обычная загрузка файла (атрибут `download`)
- Моды подписываются и скачиваются через Arma 3 Launcher после импорта пресета
