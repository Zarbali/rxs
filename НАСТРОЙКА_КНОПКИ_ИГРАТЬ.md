# Настройка кнопки «Играть»

При нажатии «Играть» открывается окно с:
- **Запустить Arma 3** — запуск игры через Steam (`steam://run/107410`)
- **Установить Arma 3** — ссылка в Steam Store (если игра не куплена)
- **Скачать пресет** — ссылка на файл пресета (появляется после настройки)
- **Моды сервера** — ссылки на Steam Workshop (появляются после настройки)

---

## Шаг 1: Настройте config.js

Откройте `config.js` и заполните `ARMA_CONFIG`:

```javascript
const ARMA_CONFIG = {
    steamRun: 'steam://run/107410',           // Оставьте как есть
    steamInstall: 'https://store.steampowered.com/app/107410/Arma_3/',
    presetUrl: 'https://ВАШ_САЙТ/RXS-preset.html',   // Ссылка на пресет
    mods: [
        'https://steamcommunity.com/sharedfiles/filedetails/?id=123456789',
        { url: 'https://steamcommunity.com/sharedfiles/filedetails/?id=987654321', name: 'Название мода' }
    ]
};
```

---

## Шаг 2: Добавьте пресет

1. Создайте пресет в **Arma 3 Launcher** (если ещё не создан)
2. Экспортируйте пресет как `.html` или `.launcher`
3. Загрузите файл в репозиторий (например, `RXS-preset.html` в корень) или на файлообменник
4. Укажите ссылку в `presetUrl`

---

## Ограничения браузера

| Действие | Возможность |
|----------|-------------|
| Запустить Arma 3 | ✅ Через `steam://run/107410` (работает при установленном Steam на ПК) |
| Установить Arma 3 | ✅ Ссылка в Steam Store |
| Скачать пресет | ✅ Ссылка на файл — пользователь скачивает и загружает в лаунчер |
| Подписаться на моды | ✅ Ссылки на Steam Workshop — подписка в браузере, скачивание через Steam |
| Автоматически установить моды в игру | ❌ Невозможно через сайт — только через Steam Workshop |

Полностью автоматическую установку Arma 3 + пресета + модов через один клик **сделать нельзя**: Steam и Arma 3 Launcher не поддерживают такую интеграцию из веба.
