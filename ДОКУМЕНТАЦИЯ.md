# RXS | Operation: Whisper of Sand — техническая документация

## Обзор проекта

Сайт RP-сервера Arma 3 с военной атмосферой, тёмной цветовой схемой и современным UI. Реализован на чистом HTML, CSS и JavaScript без сторонних фреймворков.

---

```
rxs/
├── index.html      — главная страница
├── rules.html      — страница правил игры
├── styles.css      — стили (BEM-подобная методология)
├── script.js       — логика и интерактивность
├── logo.jpg        — логотип (в навбаре)
├── server.mp4      — фоновое видео в hero-блоке

├── picture2.png    — фон секции «Присоединяйтесь»

```

## 2. Навигация (header)

**HTML:** Фиксированный `header` с `nav`, логотипом (картинка + текст «RXS»), ссылками и кнопкой мобильного меню.

**CSS:**
- `position: fixed` — навбар всегда сверху
- Градиентный фон с прозрачностью
- Класс `.scrolled` — при скролле > 50px добавляется тёмный фон и `backdrop-filter: blur`

**JavaScript:**
- `initHeaderScroll()` — отслеживает `pageYOffset`, добавляет/убирает `.scrolled`

## 3. Hero-секция (видео и контент)

### Видео-бекграунд

**HTML:**
```html
<div class="hero__video-wrap">
    <video class="hero__video" autoplay muted loop playsinline preload="auto">
        <source src="server.mp4" type="video/mp4">
    </video>
    <div class="hero__video-overlay"></div>
</div>
```

**Важно:**
- `muted` — нужен для автовоспроизведения в браузерах
- `playsinline` — для iOS
- `loop` — зацикливание

**CSS:**
- Видео: `object-fit: contain`, `transform: scale(1.55, 1.25)` — подгонка под экран
- `.hero__video-overlay` и `.hero__bg` — полупрозрачные слои для читаемости текста

### Управление звуком

**HTML:**
- Кнопка mute/unmute с двумя SVG-иконками (звук вкл/выкл)
- Слайдер `input[type="range"]` для громкости (0–100)

**JavaScript (`initHeroVideo()`):**
1. Видео стартует с `muted: true` и `volume: 0.8`
2. Клик по кнопке — переключение `video.muted`
3. Слайдер — меняет `video.volume` (0–1)
4. При громкости > 0 — звук включается, при 0 — выключается

## 4. Анимации при скролле (Scroll Reveal)

**HTML:** Элементам добавляется класс `scroll-reveal` (опционально `delay-1`, `delay-2`, `delay-3`).

**CSS:**
- Исходное состояние: `opacity: 0`, `transform: translateY(40px)`
- Класс `.revealed`: `opacity: 1`, `transform: translateY(0)`
- `transition` — плавное появление

**JavaScript (`initScrollReveal()`):**
- `IntersectionObserver` следит за видимостью элементов
- При входе в зону видимости добавляется класс `revealed`
- `rootMargin: '0px 0px -80px 0px'` — анимация запускается чуть раньше выхода элемента на экран

## 7. Мобильное меню

**HTML:** `button.nav__toggle` с тремя `span` (бургер-иконка).

**CSS:**
- На `<768px` — `.nav__links` выезжает справа (`right: -100%` → `right: 0`)
- Класс `.active` на toggle — анимация в крестик

**JavaScript (`initMobileMenu()`):**
- Клик по toggle — переключение классов `active` и блокировка скролла `body`
- Клик по ссылке — закрытие меню

---

## 8. Плавная прокрутка

**HTML:** `scroll-behavior: smooth` на `html`.

**JavaScript (`initSmoothScroll()`):** Обработка якорных ссылок `#section`, при клике — `scrollIntoView({ behavior: 'smooth' })`.

---

## 9. Адаптивность

**Брейкпоинты:**
- `768px` — навбар в режиме бургер-меню, hero padding
- `480px` — кнопки в колонку, увеличенный padding

**Отступы:** `clamp()` — адаптивные значения, например `clamp(1.5rem, 5vw, 4rem)`.

## 10. Как добавить новую секцию

1. В `index.html` после существующей секции:
```html
<section id="new-section" class="section section--new">
    <div class="container">
        <div class="section__header scroll-reveal">
            <span class="section__tag">[ 04 ]</span>
            <h2 class="section__title">Заголовок</h2>
        </div>
        <!-- контент -->
    </div>
</section>
```

2. В навбар:
```html
<li><a href="#new-section">Новая секция</a></li>
```

3. При необходимости — стили в `styles.css` для `.section--new`.
